# UniEventos

UniEventos é um protótipo acadêmico desenvolvido com o objetivo de validar a proposta de uma plataforma digital que centraliza eventos acadêmicos, oferece um sistema de recompensas e disponibiliza um espaço simples de interação entre alunos e palestrantes. O MVP foi construído utilizando HTML, CSS e JavaScript, com foco em navegação e experiência do usuário.

---

## Sobre o Projeto

O sistema funciona como uma demonstração navegável para apresentação acadêmica. As três páginas principais — início, eventos e perfil — permitem visualizar o fluxo básico da plataforma, testar filtros, navegar entre seções e observar a identidade visual proposta.

O protótipo não possui back-end; seu objetivo é validar a experiência e a proposta conceitual do sistema.

## Funcionalidades do MVP

- Listagem de eventos acadêmicos
- Filtros por categoria e instituição
- Exibição de pontos, conquistas e recompensas
- Página de perfil com eventos inscritos
- Interface simples, moderna e clara

---

## Objetivo do MVP

O MVP foi desenvolvido para validar:

- Compreensão da proposta
- Utilidade da centralização de eventos
- Clareza e atratividade do sistema de pontos
- Usabilidade e experiência geral da interface
